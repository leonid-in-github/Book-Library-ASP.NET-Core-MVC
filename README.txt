Deploy
1. Create data base in sql server.
2. Publish Book Library Data Base project to created data base.
3. Update Book_Library_EF_Core_Proxy_Class_Library.Constants.LibraryConstants.CONNECTIONSTRING 
through created data base Connection String in Book Library EF Core Proxy Class Library project. 
4. Rebuild solution.
5. Run in Visual Studio or deploy to IIS Book Library ASP.NET Core MVC project.
